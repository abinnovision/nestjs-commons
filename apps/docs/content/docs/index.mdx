`@abinnovision/nestjs-commons` is a small family of independently versioned NestJS packages.
Each package focuses on one concern and integrates cleanly with NestJS modules and DI.

## Packages

- **[`@abinnovision/nestjs-configx`](/docs/packages/configx)** — Type-safe configuration management for NestJS, powered by Standard Schema.
- **[`@abinnovision/nestjs-exceptions`](/docs/packages/exceptions)** — Entity-focused exception handling with HTTP and GraphQL support.
- **[`@abinnovision/nestjs-hatchet`](/docs/packages/hatchet)** — NestJS integration for [Hatchet](https://hatchet.run) workflow orchestration.
- **[`@abinnovision/nestjs-toolkit`](/docs/packages/toolkit)** — Shared utilities: remeda re-export, slug/sanitize helpers, UUID helpers.

## Highlights

- **Type-safe** — built for TypeScript first, with strict types and schema-driven validation.
- **Minimal surface** — each package solves one problem cleanly.
- **Standards-friendly** — Standard Schema (Zod, ArkType, …), GraphQL, and the broader Nest ecosystem.
- **Production-tested** — used in AB INNOVISION services, released under Apache-2.0 on npm and GHPR.
